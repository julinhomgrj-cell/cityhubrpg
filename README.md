# 🌆 City Hub RPG — Fã Site Oficial

Rede social para a comunidade **City Hub RPG** do Habbo Hotel Brasil.

## 📁 Estrutura
```
cityhub-rpg/
├── index.html       ← site completo
├── sw.js            ← service worker (cache PWA)
├── manifest.json    ← config PWA (instalar como app)
├── README.md
└── icons/
    ├── icon-72.png
    ├── icon-96.png
    ├── icon-128.png
    ├── icon-192.png
    └── icon-512.png
```

## 🚀 Deploy no GitHub Pages
1. Suba todos os arquivos no repositório
2. Vá em **Settings → Pages → Source → main → / (root)**
3. Clique **Save** — site disponível em `https://SEU-USUARIO.github.io/NOME-REPO`

## ✨ Funcionalidades
- Login/cadastro verificado pelo personagem Habbo (sem senha)
- Empresas com banner, logo, múltiplos cargos e membros com avatar Habbo
- Adicionar membros ao cargo por nick com preview do avatar
- Feed social com curtidas e YouTube embed
- PWA — instalável como app no celular
- Cache offline via Service Worker
- Painel admin completo com personalização do site
