# 🌆 City Hub RPG — Fã Site Oficial

Site de rede social para a comunidade **City Hub RPG** do Habbo Hotel Brasil.

## ✨ Funcionalidades

- 🔐 Login e cadastro verificados pelo personagem Habbo (sem senha)
- 🏢 Empresas com banner, logo, cargos múltiplos e sistema público/privado
- 📰 Feed social com curtidas, YouTube embed e upload de imagens
- 👤 Perfil do jogador com avatar oficial da API Habbo
- 🔔 Notificações em tempo real
- ⚙️ Painel admin com personalização completa do site (cores, logo, banner)

## 🚀 Como usar

1. Acesse o site pelo GitHub Pages
2. Clique em **Cadastrar** e verifique com seu personagem Habbo
3. Explore as empresas, publique e interaja!

## 🛠️ Tecnologias

- HTML5 + CSS3 + JavaScript puro (sem dependências)
- Dados salvos no localStorage do navegador
- API oficial do Habbo Hotel Brasil para verificação de personagens

## 🌐 Deploy no GitHub Pages

1. Faça fork ou upload deste repositório
2. Vá em **Settings → Pages**
3. Em **Source**, selecione a branch `main` e pasta `/ (root)`
4. Clique em **Save** — o site ficará disponível em `https://SEU-USUARIO.github.io/NOME-DO-REPO`
